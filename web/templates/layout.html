Welcome to TimeChange!

This project allows users to easily convert timeseries csv formatted data into fourier transform images and classify them using existing machine learning libraries without a deep knowledge of those tools.

To begin, click on the 'Load Files' tab to load and label your csv files!

Instructions

Step 1) Click the "Select Files" button to select files from the filesystem
Step 2) Type a label into the following text box
Step 3) Click the files you want to apply this label to
Step 4) Repeat steps 2 and 3 until all files are labeled
Step 5) Click "Add to Project" button to add these labeled files to the project
